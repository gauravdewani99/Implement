# Implement

A client onboarding configuration tool built for ReBound Returns. Streamlines the setup process for new clients joining the returns platform.

## What it does

- Manages the end-to-end client onboarding workflow
- Provides configuration management for client-specific settings
- Stores onboarding data persistently in PostgreSQL

## Tech stack

- **Backend:** Node.js + Express
- **Frontend:** React
- **Database:** PostgreSQL
- **Deployment:** Docker + Docker Compose
